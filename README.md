# QBMS v2.0
Question Bank Management System merged with LMS for question storage and mock generation.

## Getting Started

### Prerequisites
- Node.js installed

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure Environment:
   Ensure you have a `.env` file with the necessary API keys (Supabase, etc.).

### Running the App
Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
